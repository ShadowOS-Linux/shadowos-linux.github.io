Custom Bazzite image with Cosmic/Gnome/KDE/XFCE.

FEATURES

Atomic Foundation

Inherits all the cutting-edge game compatibility, stability, and immutability benefits of Bazzite, Universal Blue, and Fedora Atomic.

4 Desktop Environments

Choose between COSMIC, XFCE, KDE, or GNOME out of the box instead of being restricted to just two layout choices.

Nostalgic Themes

Features a seamless Windows 7 look in KDE via AeroThemePlasma and a tweaked Windows XP environment in XFCE using xfce-winxp-tc.

Tailored Layouts

GNOME comes pre-configured with active production extensions, COSMIC features a familiar Windows-style panel configuration, and stays up to date via the cosmic-epoch copr repository.

Privacy-First Browsing

Firefox is swapped for LibreWolf with user-friendly settings applied, styled dynamically with the Firefox GNOME theme and compact extension panels via firefox-csshacks.

Upgraded Gaming Stack

Steam is completely optional. Features automatic DLSS generation upgrades across all titles and includes a custom build of Heroic Games Launcher containing the Console Mode fix.

Android Power Tools

Full system integration for mobile power users with ADB, scrcpy, and the Universal Android Debloater bundled directly into the core image.

Smart Automation

Toggle hardware integrations effortlessly with built-in commands like ujust scrcpy-cam to use your phone as a webcam, ujust toggle-autologin for auto-login, and global MangoHUD toggling via the END key.
